# Unscramble

유어슈 안드로이드 과제

뒤죽박죽 되어 있는 스펠을 보고 원래 단어를 맞추는 게임

## 목표

YDS 사용해보기

클린 아키텍처 적용해보기

- Step 1. String array resource
- Step 2. Http 클라이언트 라이브러리로 불러오기 (Retrofit2)
- Step 3. 타이머 구현

## 팀원

- Gio [Android]
- Bona [Android]
- Wes [Android]
